# Web Test Security Demo

Webテストにおける不正行為（カンニング）検知の概念実証（PoC）デモです。
ブラウザの標準機能のみを使用して、以下の不正操作を検知・記録します。

## 機能 (Features)

*   **フォーカス検知 (Focus Tracking)**:
    *   タブの切り替え（`visibilitychange`）
    *   ウィンドウのフォーカス喪失（`blur` / `focus`）
    *   これらを「ハイブリッド検知（イベント + ポーリング）」で確実に捕捉します。
*   **高精度タイマー (High-Precision Timer)**:
    *   タイムスタンプ差分方式により、長時間のテストでも正確な「アクティブ時間」と「非アクティブ時間」を計測します。
*   **キー入力記録 (Key Logging)**:
    *   全てのキー入力を記録します。
    *   `PrintScreen` (スクリーンショット試行) や `F12` (開発者ツール) などの怪しいキー操作を警告として記録します。
*   **3カラムログ (3-Column Logs)**:
    *   **Focus History**: フォーカス移動の履歴
    *   **System Alerts**: システム警告（開始/停止、スクショ検知など）
    *   **Key History**: キー入力の生ログ

## ファイル構成 (File Structure)

このプロジェクトは以下の3つのファイルで構成されています。これらは全て同じフォルダに配置する必要があります。

*   `index.html`: アプリケーションの構造とレイアウト
*   `style.css`: デザインとスタイル定義
*   `main.js`: 検知ロジックとタイマー処理

## 使い方 (Usage)

1.  `index.html` をブラウザで開きます。
2.  **「監視開始」** ボタンをクリックします。
3.  別のタブを開いたり、別のウィンドウ（メモ帳など）に切り替えたりして、検知されるか試してください。
4.  キーボードで入力を行い、ログが記録されるか確認してください。
